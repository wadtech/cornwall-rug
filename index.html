Welcome to the Cornwall Ruby Users Group.

We're a group of hobbyists and professionals who use and, presumably, like ruby, the programming language. This user group is a way to meet other people in the industry locally with Ruby experience and to learn from each other.

We use Slack to share news, tips and chat about the ruby programming language.

Join us!

If you're already a member of the workspace find us at https://cornwallruby.slack.com

What is Ruby?

Ruby is a programming language designed by Yukihiro Matsumoto which is used for many applications around the world, from native mobile to web to server and infrastructure.

A dynamic, open source programming language with a focus on simplicity and productivity. It has an elegant syntax that is natural to read and easy to write.

Do I need to be a ruby programmer to join?

Definitely not! We cater for members of all ability, from curious to hobbyist to professional alike.

Do you only talk about Ruby?

Ruby usually goes hand-in-hand with other technologies or as part of a wider ecosystem in tools and applications. As a result we often talk about other technologies and their uses.

Also, being a social group there are plenty of other things to talk about!
